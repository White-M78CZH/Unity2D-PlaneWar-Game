# Unity2D-PlaneWar-Game
> Unity 2D 竖版飞机大战 - 应届生独立开发项目

---

## 🎮 项目简介
这是一个基于 Unity 引擎开发的 2D 竖版射击游戏，是我作为软件工程专业应届生的独立作品。项目完整实现了玩家控制、敌机生成、道具系统、碰撞检测与本地最高分存储等核心玩法，是一次完整的游戏开发实践。

## 🛠️ 技术栈
- **引擎**：Unity 2021.3 LTS
- **语言**：C#
- **核心系统**：对象池、事件系统、数据持久化

## 🎯 核心功能
- ✅ 玩家飞机控制与移动边界限制
- ✅ 多种敌机AI与弹幕生成系统
- ✅ 道具拾取与效果系统（生命、子弹升级）
- ✅ 碰撞检测与伤害反馈
- ✅ UI系统（生命值、分数、游戏结束界面）
- ✅ 本地最高分存储（PlayerPrefs实现）

## 🚀 如何运行
1. 克隆或下载本项目到本地
2. 使用 Unity 2021.3 或更高版本打开项目
3. 在 Unity 编辑器中打开 `Assets/Scenes/Main.unity` 场景
4. 点击运行按钮即可体验游戏
