# Unity2D-PlaneWar-Game
> 手绘风2D竖版飞机大战 | 应届生独立开发完整游戏项目

---

## 🎮 项目简介
这是一款基于 Unity 引擎独立开发的手绘风格2D竖版射击游戏，为软件工程专业应届生个人开发作品。
项目覆盖主菜单、战斗、暂停、结算完整游戏状态流程，搭建了阶梯式难度成长体系与多类型道具系统，支持PC/移动端双端运行，完成了多分辨率自适应适配与全流程玩家体验优化，是一次完整的游戏客户端开发实践。

## 🛠️ 技术栈
- **开发引擎**：Unity 2021.3 LTS
- **开发语言**：C#
- **设计思想**：面向对象编程、单例模式、模块化开发
- **核心系统**：2D物理碰撞、UGUI布局、动画状态机、协程、数据持久化
- **工程能力**：Git版本管理、多平台打包适配、基础性能优化

## 🎯 核心功能
### 战斗玩法系统
- ✅ 玩家双端操控：支持PC鼠标拖拽/移动端触摸操作，包含移动边界限制、受击无敌闪烁、护盾防护机制
- ✅ 多类型敌机：3种差异化敌机，配套独立动画与音效，随游戏等级递进解锁刷新
- ✅ 多样化射击：玩家射速随等级成长，敌机支持三向射击、子弹反弹等攻击模式
- ✅ 阶梯难度：敌机血量、移速、刷新频率随等级动态提升，平滑难度曲线

### 道具与成长系统
- ✅ 4类分级解锁道具（生命恢复、火力升级、护盾防护、全屏清屏）
- ✅ 道具拾取反馈与专属音效，强化玩家正反馈体验
- ✅ 等级成长机制，逐步丰富玩法深度

### UI与多端适配
- ✅ 完整UI体系：主菜单、战斗HUD、暂停面板、结算界面
- ✅ 多分辨率适配：Canvas锚点自适应方案，兼容各类竖屏设备，解决长屏UI错位问题
- ✅ SDF中文字体：替换动态字体方案，修复字体模糊与缺字异常
- ✅ 新手引导：开局引导、道具首次提示，优化玩家上手体验

### 工程与系统架构
- ✅ 全局单例音频管理器，实现跨场景BGM无缝切换与音效统一调度
- ✅ 基于Sprite Bounds的无缝背景滚动方案，解决长屏背景露边与滚动累积偏移问题
- ✅ 本地数据持久化：最高分存储、存档清除功能
- ✅ 完整游戏状态管理：开场倒计时、暂停、结算重开全流程闭环

## 🚀 如何运行
1. 克隆或下载本项目到本地
2. 使用 Unity 2021.3 LTS 及更高版本打开项目
3. 在 Unity 编辑器中打开 `Assets/Scenes/Game.scene` 场景
4. 点击运行按钮即可体验游戏
